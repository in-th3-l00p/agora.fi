# branding & identity

## name

**AGORAFI** (pronounced: ah-GOR-ah-fie)

- **Agora** (Greek: agorá) — the ancient public square, marketplace, and assembly place
- **Fi** (from DeFi) — decentralized finance

Together: *"The financial public square"* — where communities gather to build decentralized economies.

### usage rules

- Always written in all caps: **AGORAFI**
- Never: Agorafi, agorafi, AgoraFi
- Optional delta symbol variation: **AGOR∆FI**
- Subtle underline or geometric accent on the logo

## visual identity

**Aesthetic: "Neo-Classical Cyberpunk"** — a blend of ancient Greek architecture with futuristic technology. Think marble columns with neon glow, isometric cityscapes with glass morphism overlays.

**Style keywords:** Isometric 3D, glass morphism, neon accents, data-driven, abstract grid patterns.

### design language

- **Glass morphism** — translucent cards with frosted-glass effects
- **Neon glow** — subtle light emission on interactive elements and key metrics
- **Isometric perspective** — consistent 3D angle across all spatial visualizations
- **Grid patterns** — data-driven, silicon-valley-inspired abstract backgrounds
- **Dark-first** — the interface is designed for dark mode as the primary experience
