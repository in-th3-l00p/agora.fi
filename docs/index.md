# AGORAFI

**"Where Communities Build Economies"**

AGORAFI is a decentralized platform for creating autonomous virtual spaces where communities build their own economies. It combines NFT-based digital real estate with DeFi mechanics and AI-powered content generation to create self-sustaining community ecosystems.

Users purchase **Living Tiles** — NFT properties that own assets, generate yield, earn rental income, and govern themselves autonomously. Each tile is not just a collectible — it's a programmable economic entity that works for its owner 24/7.
